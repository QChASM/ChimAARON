substitute

Usage:
substitute substituent [ form format ]

substitute replaces a functional group that starts on the specified atom with a new substituent. The format of substituent is passed to form:

  • from_library (default) - substituent is loaded from the AaronTools library
  • iupac - substituent is grabbed from the CACTVS web API after the IUPAC name is parsed by OPSIN web API
  • smiles - substituent is grabbed from the the CACTVS web API
replace causes the structure being modified to close before the new structure is opened.
