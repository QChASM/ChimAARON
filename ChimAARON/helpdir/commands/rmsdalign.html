rmsdalign

Usage:
rmsdalign [ sort ] [ longsort ]
rmsdalign atom-spec [ sort ] [ longsort ]

rmsdalign aligns the molecules to which the atom in atom-spec belong to minimize the RMSD between the atoms in atom-spec. If no atoms are specified, the current selection is used. The keywords sort and longsort should be added if you'd like AaronTools' RMSD alignment subroutine to sort the atoms to produce a better alignment. Longsort specifies a more robust and more time-consuming sorting algorithm. Sort specifies a faster algorithm.
