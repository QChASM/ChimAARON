closering

Usage:
closering ring form format [replace true|false]

closering closes an incomplete ring. Two atoms must be selected for closering to attempt to turn these atoms and the atoms between them into a ring ring. The format of ring is specified by form. Valid keywords for form are:

  • from_library (default) - ring fragment is loaded from the AaronTools library
  • iupac - ring fragment is grabbed from the CACTVS web API after the IUPAC name is parsed by OPSIN web API
  • smiles - ring fragment is grabbed from the the CACTVS web API
replace causes the structure being modified to close before the new structure is opened.
