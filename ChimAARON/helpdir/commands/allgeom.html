allgeom

Usage:
allgeom path/to/some.file [ roundTrip true | false ] [ dynamicBonds true | false ] [ drawMode Stick | Wire ]

allgeom loads the specified file as a movie. If roundTrip is true, the loaded animation will play from the first coordinate set to the last, and then back to the start before the movie player resets to the first frame. If dynamicBonds is true, all arguments will be passed to ChimAARON.getDynamicBondScript(). The movie's script will then be set to the output of this subroutine. This should result in an an animation where bonds and coordination pseudobonds are broken and formed dynamically.
