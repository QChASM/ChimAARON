follow

Usage:
follow filename [ nFrames n ] [ roundTrip true | false ] [ maxDisplacement m ] [ mode v ]

follow loads normal mode v from the specified file. The keywords modify certain settings of the animation:

  • nFrames (default 61) - number of frames in the animation
  • roundTrip - true if the mode is animated "round-trip" (i.e. structure goes back and forth through equilibrium before the movie player resets)

  • mode (default 1) - normal mode to animate (1-indexed)
