arnscl

Usage:
arnscl

arnscl sets the label on atoms for all open models to match AARON's relative numbering. Atoms on the substrate are labeled Sn, where n is the relative atom number. Likewise, atoms on the center and ligand are labeled Cn and Ln, respectively. Note that although multiple centers may be labled for the same molecule, AARON currently only supports single-center structures.

Labels are determined by reaading the comment line of the file for an open model (Chimera sets the name of molecules opened from .xyz files to the comment line, if there is one). If no center is found in the comment or the comment doesn't appear to be in AARON's format, arnscl will look for transition metals instead. The first transition metal found will be marked as the center; all atoms before the center are marked as substrate, and all after are marked as ligand.
