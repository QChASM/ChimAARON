freqvec

Usage:
freqvec filename [ color color ] [ scale s ] [ mode v ] [ massWeight true | false ]

freqvec loads the structure in the specified file and represents normal mode v using vectors. Keywords modify the vectors:

  • color (default green) - color of the vectors (either a named Chimera color or r,g,b[,a] list)
  • scale (default 1.5) - length of the longest vector (in Ã…); using a negative value will flip the direction of the vectors
  • mode (default 1) - mode to display vectors for (1-indexed)
