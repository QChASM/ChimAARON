tsbond

Usage:
tsbond [ selec atom-spec ] [ color color ] [ transparency precent ]

tsbond deletes the selected bonds and puts transparent ghost atoms on top of the previously bonded atoms. These ghost atoms are not displayed (unless show or display is run), but the bond between them is.
Alternatively, if two atoms are specified, bonded ghost atoms are placed on top of these and the bond between the selected atoms is deleted (if they are bonded).

Ghost atoms are named 'ghost', allowing them to be selected or deselected with the command line specified '@ghost'.

The keywords modify the appearance of these bonds:

  • color (default byatom) - color of bonds; may be Chimera named color, r,g,b[,a] tuple, or byatom.
    NOTE If you don't want to color the bonds byatom, consider coloring the bond and setting its transparency with the color and transparency Chimera commands. There is no benefit to using tsbond with a specified color over the built in Chimera commands.
  • transparency (default 50) - percent transparency of the bonds
