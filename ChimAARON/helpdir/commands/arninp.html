arninp

Usage:
arninp record name model selection
arninp name mapligand ligand atom selection
arninp name substitute substituent atom selection
arninp name write [keyword value [keyword value]]

arninp is the command line implementation of the AARON Input Recorder GUI.

When passed the record argument, a new record is created, with the involved models specified by the model selection.

When mapligand is passed, the ligand (key atoms specifed in the atom selection) is swapped for the specified ligand. This ligand change is also recorded under name.

When substitute is passed, the substituent is added at the position(s) specifed in the atom selection. This is also recorded under name.

When write is passed, the specified recording is printed to the reply log. AARON input keywords can also be specified.
