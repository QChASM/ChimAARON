Welcome to the ChimAARON User's Guide! Here, you can find tutorials on using the functionality added in the ChimAARON plugin. You can also find details about ChimAARON tools and commands. Many of the features added in this plugin are built off of our AaronTools Python module. Information about AaronTools and AARON can be found on the QChASM GitHub page, or on the QChASM website.

ChimAARON aims to add features that are useful for setting up AARON, and modifying and representing small systems that are studing using quantum mechanical methods.
