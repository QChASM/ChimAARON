ChimAARON Commands

Welcome to the ChimAARON Commands! Here, you can find information on the commands added by ChimAARON.

  • allgeom - load all structures in a file as an animation
  • arninp - set up an AARON input file
  • arnscl - set atom labels to correspond to AARON's relative numbering for substrate, center, and ligand
  • closering - change selected atoms for part of a ring
  • follow - load an animation for a normal mode from a file
  • freqvec - display normal mode displacement vectors given a file
  • mapligand - swap a ligand
  • printxyz - print atom selection to reply log in xyz format
  • rmsdalign - align atoms in different models to each other
  • substitute - swap a functional group
  • tsbond - replace selected bonds with transparent bonds
