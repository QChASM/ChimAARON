Adding to Your Personal Library

AaronTools is packaged with a number of ring fragments, ligands, and substituents in its built-in library. These ligands, substituents, and rings power AaronTools'/ChimAARON's structure modification tools, like substitute and Modify Structure. If AaronTools doesn't have a ligand or substituent you use frequently, simply add it to your personal library. Here, you'll find information about adding more items to your personal AaronTools library.

Structures in your personal library are stored locally. For information on specifying the location of your personal library directory, see tutorial on setting up AaronTools.

The tool for adding structures to your personal library can be opened under Tools → ChimAARON → Add to AaronTools Library.

Adding a Ligand

To add a ligand, open a molecule with that ligand already on it. Select the molecule from the table. Without entering a name, press Add to library. This will open the ligand in a new model. The ligand's key atoms (the ones that coordinate a transition metal) should be outlined in blue.

If the ligand looks fine, go back to the tool and enter a name for the ligand (select the full molecule, not the model that is just the ligand).

To replace a ligand with the same name, check the Overwrite existsing box.

Adding a Ring Fragment

To add a ring fragment, open a full structure for that ring (e.g. benzene, pyridine, furan). The Walk atoms define the direction AaronTools goes around the ring when it is adding the ring to a structure. Without entering a name, press Add to library. This will open the ring in a new model. Arrows will point from one 'walk' atom to the next.

If the ring looks fine, enter a name and press Add to library again.

Adding a Substituent

To add a substituent, open a molecule with that substituent. Select all of the atoms on that substituent and press current selection. Enter the conformer information for the substituent. Without entering a name, press Add to library. This will open a new model with just the substituent. In this new model, the bond connecting the substituent to the rest of the molecule should be represented with a semi-transparent cylinder.

If the substituent looks fine, enter a name and press Add to library again.
