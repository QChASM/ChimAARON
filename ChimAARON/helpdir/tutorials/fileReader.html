Opening Files through the AaronTools FileReader

ChimAARON enables several file types to be opened in Chimera by passing them through AaronTools' FileReader. This includes Gaussian .com files and .log files. To open such files using the File → Open... option, you can specify that the File type is an AaronTools coordinate file. This is not necessary if no other Chimera-readable file has a .com or .log extension, as Chimera just uses the first thing it finds that can open a given file type.

Similarly, the open command can be used to open AaronTools-readable files. The AaronTools: prefix can be used to ensure the file is parsed by AaronTools and not some other Chimera file reader: 

open AaronTools:path/to/file.log

Like before, using prefixes is not necessary if no other Chimera-readable filetype has a .com or .log extension.
