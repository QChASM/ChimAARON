Visualizing Normal Modes

ChimAARON makes it easy to visualize normal vibrational modes in Chimera. In order to load the normal modes of a molecule, ChimAARON requires a Gaussian .log file containing frequency information is required.

To load the frequency information, go to the Visualize Normal Mode... option under Tools → ChimAARON. You will then be prompted to locate your frequency file. Once the file is selected, a dialog menu will open up listing the normal modes. These modes may be sorted by frequency or IR Intensity. A mode can be selected from the table and visualized by pressing the Display mode button.

There are two ways to visualize a normal mode: displacement vectors, and an animation.

When a normal mode is displayed using vectors, you can change several things about the vectors:

  • Vector scaling - change the length of the longest vector
  • Use mass-weighted - whether or not to remove mass-weighting from the vectors
  • Vector color - the color of the vectors

When loading the mode as an animation, you have different settings:

  • Max. displacement - most any atom moves from its equilibrium position
  • Number of frames - how long the animation is

These tools can also be accessed through the command line. See the documentation on follow and freqvec for details.
