Setting up AaronTools

Almost all of the functionality added by ChimAARON is based on our AaronTools Python 3 module. The AaronTools installation that comes with ChimAARON is compatible with Python 2, which is what Chimera uses.

Adding the ChimAARON Plugin

NOTE - you may experience some issues using ChimAARON if AaronTools is installed in a cloud-synced directory, like a Dropbox folder.

  1. To add ChimAARON as a Chimera plugin, go to Tools → Additional Tools → Add third-party plugin location...
    • This will open a dialog menu, in the bottom half of which you can specify the location of plugin folders
  2. Press Add... and navigate your way to the directory containing your ChimAARON and AaronTools directories (e.g. 'plugins' for /home/CoolUser/plugins/AaronTools and /home/CoolUser/plugins/ChimAARON)
    • Pressing Open will add that location to the "Third-party plugin locations" box
    • Check to make sure the location is set correctly (for our CoolUser example, you'd see "• plugins - /home/CoolUser" or "• /home/CoolUser/plugins" depending on a setting in your General Preferences)
    • Make sure to press Save
Chimera is now able to discover AaronTools and ChimAARON. Now, we just need some AaronTools dependencies.

Installing Dependencies

If you installed Chimera in a directory that requires administrator/root privileges to modify, you'll need to run Chimera as an administrator/super user for this.

AaronTools uses two Python modules that don't come with Chimera: `cclib` and `future`. If you're comfortable installing those manually, feel free to do so. A simpler option is to go through ChimAARON. Go to Tools → ChimAARON → Set up AaronTools. This menu option will go away if ChimAARON does not run into any errors when importing AaronTools.

You will see a dialog menu with buttons that simplify the setup process.

ChimAARON utilizes pip to install the AaronTools dependencies, so first you'll want to press install pip with ensurepip and update pip. Check Chimera's reply log to make sure pip was installed and/or updated to the newest version.

Once you've got pip, press install dependencies to install `cclib` and `future`.

Backporting AaronTools

If you are using the AaronTools installation that comes with ChimAARON, you should not need to do this

We use the pasteurize script that comes with `future` to fix most of AaronTools' compatibility issues between Python 2 and 3. To run this script on AaronTools, press pasteurize AaronTools. Monitor Chimera's reply log to make sure this finishes without issue.

Once pasteurize finishes, we still need to fix a couple of things that pasteurize doesn't catch. Simply press fix misc. backporting.

Now you should just have to restart Chimera for all of the ChimAARON tools to start functioning!

Personal Library

To specify the location of you personal AaronTools library, press set AARONLIB location. This will open up a directory explorer. Navigate to you personal library directory and select it. You will have to restart Chimera to be able to use things from your personal library.
